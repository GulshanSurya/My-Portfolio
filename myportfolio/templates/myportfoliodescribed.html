My Portfolio What is "Myportfolio"?

The Project named "Myportfolio" is a basically a django website which briefly describes about the projects, internships and other technical and non-technical activities that are done by me (Gulshan Surya). As we already know, this website is created using HTML and CSS for front-end and uses very basic python/Django framework for back-end.

What functionalities does "Myportfolio" consists?
Talking about the home section
  1. In the home page of my 'myportfolio' project, I have given a navigation bar which has some buttons on it having relation to different addreses to mine to different platforms.
  2. The navbar background has been created by using "div" tag in html and buttons on it are not actually buttons but are created by using "ul" and "li" i.e. unordered list and list tags respectively. When user makes a click on any of the list items accept for "about me", the user's browser is redirected to the profile page of mine of the respective platform. And when the user clicks on "contacts/phone"section shown in the form of a button, there is an API which takes the user to my whatsapp number which is also my contact number.
  3. In addition, the home page consists of three buttons aiming to divide the portfolio sections into three main categories i.e projects, internships and other activities done by me.These buttons are created using "button" tag unlike navigation bar buttons.
Talking about the My Project section
  1. "My Projects" section that lists out the different domains of projects.
  2. In this section, again there are four buttons made using "button" tag.
  3. On clicking the "Django" named button, the browser is redirected to a page which consists of a table that lists out and shortly describes the projects made using Django.
  4. On clicking the "Python" named button, the browser is redirected to a page which consists of a table that lists out and shortly describes the projects made using Python.
  5. On clicking the "AI/ML" named button, the browser is redirected to a page which consists of a table that lists out and shortly describes the projects made using Artificial Intelligence or Machine Learning.
  6. On clicking the "Frontend" named button, the browser is redirected to a page which consists of a table that lists out and shortly describes the projects made using front-end technologies (in most cases HTML, CSS and JavaScript).
